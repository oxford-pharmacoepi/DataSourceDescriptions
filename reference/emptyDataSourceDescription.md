# Create an empty `data_source_description` object

Create an empty `data_source_description` object

## Usage

``` r
emptyDataSourceDescription()
```

## Value

An empty `data_source_description` object.

## Examples

``` r
emptyDataSourceDescription()
#> 
#> ── 0 data source descriptions ──────────────────────────────────────────────────
#> 
```
