# Export data source descriptions

Export data source descriptions

## Usage

``` r
exportDataSourceDescription(x, path, type = "json")
```

## Arguments

- x:

  A data source description.

- path:

  Directory where files will be written.

- type:

  File type to export. Supported values are `"json"` and `"csv"`.

## Value

Invisibly returns the exported file paths.
